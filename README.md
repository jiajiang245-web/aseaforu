# A Sea for You

Static GitHub Pages site for an NFC-based sound archive.

## Deploy

1. Copy the audio files into `audio/` using the filenames listed in `nfc-map.csv`.
2. Publish this folder as the root of a GitHub Pages site.
3. Program NFC tags with the clean paths, for example `/venice01/`.

Each sound page is a directory with its own `index.html`, so GitHub Pages serves clean URLs without a backend.
